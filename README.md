# Banking-Institution-Analysis
Python Project for University (Sci-Kit-Learn, Pandas)

In this project, we worked with a Portuguese banking dataset to build classification models that predict whether a client would subscribe to a term deposit based on demographic and campaign-related attributes.

### Tools & Techniques:

- **Python**, **Jupyter Notebooks**, **Scikit-learn**, **NumPy**
- Models: **Decision Tree**, **Logistic Regression**, **K-Nearest Neighbors (KNN)**
- Preprocessing: **Label Encoding**, **Ordinal Encoding**, and **Missing Value Removal**

### What I Did:

- Cleaned and prepared the dataset by removing rows with unknown values and transforming categorical attributes into numerical format using label encoders.
- Built a **Decision Tree Classifier**, tuned `max_depth` to avoid overfitting, and achieved an accuracy of **93.67%** on the test set.
- Implemented **Logistic Regression** for binary classification of the target attribute (`yes`/`no`) and achieved **89% accuracy**.
- Applied a **K-Nearest Neighbors (KNN)** model with optimized parameters, achieving **92% accuracy** using 3 neighbors.
- Evaluated all models using **classification reports** (accuracy, precision, recall, F1-score) to compare performance.
- Visualized the decision tree using **Graphviz** to enhance interpretability.

This project demonstrated my ability to clean real-world data and evaluate them using standard machine learning metrics, all within Python’s data science ecosystem.
